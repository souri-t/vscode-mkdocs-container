# API-004 タスク更新

## 概要

| 項目 | 内容 |
|------|------|
| メソッド | `PATCH` |
| パス | `/tasks/{taskId}` |
| 説明 | タスク情報を更新する |

## リクエストボディ例

```json
{
  "title": "牛乳とパンを買う",
  "status": "completed",
  "priority": 1,
  "dueAt": "2026-04-26T18:00:00+09:00"
}
```

## レスポンス例

```json
{
  "taskId": "TSK-000123",
  "result": "updated"
}
```

## 備考

- 部分更新を前提とする
- 更新対象が存在しない場合は `404 Not Found`
