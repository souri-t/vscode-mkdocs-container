# API-002 タスク詳細取得

## 概要

| 項目 | 内容 |
|------|------|
| メソッド | `GET` |
| パス | `/tasks/{taskId}` |
| 説明 | 指定したタスクの詳細情報を取得する |

## パスパラメータ

| パラメータ | 必須 | 型 | 説明 |
|------------|------|----|------|
| `taskId` | Yes | string | タスク ID |

## リクエスト例

`GET /tasks/TSK-000123`

## レスポンス例

```json
{
  "taskId": "TSK-000123",
  "title": "牛乳を買う",
  "description": "仕事帰りにスーパーで購入する",
  "status": "open",
  "priority": 2,
  "dueAt": "2026-04-26T18:00:00+09:00",
  "reminderAt": "2026-04-26T17:00:00+09:00",
  "labels": [
    {
      "labelId": "LBL-001",
      "labelName": "買い物"
    }
  ],
  "createdAt": "2026-04-24T20:00:00+09:00",
  "updatedAt": "2026-04-25T09:30:00+09:00"
}
```

## 備考

- 指定したタスクが存在しない場合は `404 Not Found`
