# 付録

## 用語集

**JWT**  
JSON Web Token の略。認証情報を表現するトークン形式。

**Bearer Token**  
HTTP ヘッダーで送信する認証トークン。

**Room**  
Android アプリで利用するローカル DB ライブラリ。

**PATCH**  
一部項目のみを更新する HTTP メソッド。

## 非機能要件

| 項目 | 内容 |
|------|------|
| タイムアウト | 15秒 |
| 監査ログ保管期間 | 365日 |
| 可用性目標 | 99.9%以上 |
| 最大同時接続数 | 300 |
