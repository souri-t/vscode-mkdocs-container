# 画面設計

## 画面一覧

| 画面ID | 画面名 | 用途 |
|--------|--------|------|
| S001 | ログイン画面 | ユーザー認証 |
| [S002](s002-task-list.md) | タスク一覧画面 | 未完了、完了済みタスクの表示 |
| S003 | タスク詳細画面 | タスク詳細の参照、編集 |
| [S004](s004-task-create.md) | タスク登録画面 | 新規タスクの入力 |
| S005 | ラベル管理画面 | ラベルの登録、編集 |
| S006 | 設定画面 | 通知、同期、ログアウト設定 |

## 画面遷移図

```plantuml
@startuml
[*] --> ログイン画面
ログイン画面 --> タスク一覧画面 : 認証成功
タスク一覧画面 --> タスク詳細画面 : タスク選択
タスク一覧画面 --> タスク登録画面 : 追加ボタン
タスク一覧画面 --> ラベル管理画面
タスク一覧画面 --> 設定画面
タスク登録画面 --> タスク一覧画面 : 保存
タスク詳細画面 --> タスク一覧画面 : 更新 / 削除
@enduml
```
