# 外部仕様書 - 概要

**アプリ名:** ToDo App  
**バージョン:** 1.0  
**改訂日:** 2026-04-25

## 目的

本書は、Android 向け ToDo アプリの外部仕様を整理し、設計、実装、試験に必要な共通認識を作ることを目的とする。サンプルドキュメントとして、必要最低限の構成と記述例を示す。

## システム概要

本アプリは、個人利用を想定した ToDo 管理アプリである。ユーザーは Android 端末からタスクの登録、編集、完了、削除を行える。アプリはローカル DB にデータを保持し、オンライン時は API サーバーと同期する。

## 想定利用者

- 日々の作業や買い物メモを管理したい一般ユーザー
- オフライン環境でもタスク確認を行いたいモバイル利用者

## 対象範囲

本仕様書で扱う機能は以下の通り。

- ログイン機能
- ToDo タスク一覧表示
- ToDo タスク登録、更新、削除
- タスク完了状態の切り替え
- 期限日、優先度、ラベルによる管理
- ローカル保存とサーバー同期

## 前提条件

- クライアントは Android 14 以降を対象とする
- 認証済みユーザーのみ API 同期を利用できる
- 通知は Android 標準通知機能を利用する
