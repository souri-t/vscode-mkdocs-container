# 付録

## 用語集

**Room**  
Android 向けの SQLite ラッパーライブラリ。

**FCM**  
Firebase Cloud Messaging。プッシュ通知配信に利用する。

**同期**  
ローカル DB とサーバー DB のデータ差分を整合させる処理。

**リマインド通知**  
期限到来前にユーザーへ作業を知らせる通知。
