# 付録

## 用語集

**JWT**  
JSON Web Token の略。認証情報を表現するトークン形式。

**Bearer Token**  
HTTP ヘッダーで送信する認証トークン。

**Idempotency**  
同一リクエストを複数回実行しても結果が変わらない性質。

## 非機能要件

| 項目 | 内容 |
|------|------|
| タイムアウト | 30秒 |
| 監査ログ保管期間 | 365日 |
| 可用性目標 | 99.9%以上 |
| 最大同時接続数 | 500 |
