# 付録

## 用語集

**API**  
Application Programming Interfaceの略。アプリケーション間のインターフェース仕様。

**JWT**  
JSON Web Tokenの略。認証トークンの標準形式。

**CRUD**  
Create（作成）、Read（参照）、Update（更新）、Delete（削除）の頭文字。データの基本操作を指す。

**REST**  
Representational State Transferの略。Web API設計の原則。

**バリデーション**  
入力データの妥当性チェック。必須チェック、形式チェック、範囲チェックなど。

**ページネーション**  
大量データを複数ページに分割して表示する仕組み。

**セッション**  
ユーザーのログイン状態を保持する仕組み。
