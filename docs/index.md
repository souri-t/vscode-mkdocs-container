# システム仕様書へようこそ

このドキュメントは、MkDocsを使用して作成されたシステム仕様書のサンプルです。
VS CodeのDevContainer環境で構築されており、PlantUMLによる図の作成も可能です。

## プロジェクト概要

本プロジェクトは、保守性の高いドキュメント作成環境を提供することを目的としています。

### 特徴
- **Markdown形式**: テキストベースで履歴管理が容易
- **PlantUML対応**: コードからシーケンス図やクラス図を生成
- **VS Code連携**: DevContainerにより、環境構築が不要
- **日本語対応**: 日本語検索やUIの日本語化に対応
